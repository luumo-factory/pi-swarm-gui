# Pi Swarm GUI

A Swing-based desktop GUI for Pi Swarm.

## Requirements

- Java 25 (or newer)
- Maven 3.9+

## Build

```bash
mvn clean package
```

## Run

```bash
mvn exec:java
```

Or run the packaged jar:

```bash
java -jar target/piswarm-gui-0.1.0-SNAPSHOT.jar
```

## Test

```bash
mvn test
```
